# Student Management System (Dart)
This is a simple Dart program that allows the user to input student information, store it, calculate grades, and display sorted results.


# Features:
- Input student name, ID, and score
- Ensures unique ID for each student
- Automatic grade calculation (A+ to F)
- Sorts students by score (High â†’ Low)
- Shows total count, highest and lowest score


# How to Run This Project

1. Make sure Dart SDK is installed on your computer.
   Download link: https://dart.dev/get-dart

2. Open a terminal or Git Bash window.

3. Navigate to the folder where the file is saved. Example:
   cd "D:/flutter projects/student_management"

4. Run the program using the following command:
   dart run student_records.dart

5. The program will ask for:
   - Student Name
   - Student ID
   - Student Score

6. To stop entering student data, type:
   exit

7. After exiting, the program will display:
   - Sorted student list (High and Low score)
   - Total number of students
   - Highest and lowest scores
